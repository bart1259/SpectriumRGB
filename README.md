# Spectrum RGB

Excel macro for setting the background of cells into the color corresponding to the wavelength (in nanometers) in the value of the cell.

<img src="https://user-images.githubusercontent.com/21147581/148306885-1ee18a39-478e-480f-af9f-d4ff3e15b5cc.png" alt="drawing" width="700"/>

## How to install
Use the provided xlsm spreadsheet or install it yourself!

1. In Excel options, make sure the developer ribbon is enabled.

<img src="https://user-images.githubusercontent.com/21147581/148305662-d8c8c07b-7d22-479b-83a4-b19d007e0e43.png" alt="drawing" width="700"/>

2. Now under the Developer Ribbon select Macros and type "SpectrumRGB" and click "Create"

<img src="https://user-images.githubusercontent.com/21147581/148305767-c895eadd-3781-49c4-a159-eecf16564bf1.png" alt="drawing" width="700"/>

3. Microsoft Visual Basic for Applications should open. Paste the contents of macro.txt into the Module1 section

<img src="https://user-images.githubusercontent.com/21147581/148305881-9f1bcd77-f2a9-4013-9dce-db3e6692bae4.png" alt="drawing" width="700"/>

4. Save (ctrl + s) and save the workbook as a an Excel Macro-Enabled Workbook (.xlsm)
5. Exit out of Microsoft Visual Basic for Applications
6. Macro has been succesfully !

## How to run

1. Select the contents you wish to change the background of

<img src="https://user-images.githubusercontent.com/21147581/148306609-bafd6b0b-cfd6-48d9-b75e-0303bc8d01b9.png" alt="drawing" width="700"/>

2. Run the macro (Developer > Macros > Select "SpectrumRGB" > Run)

<img src="https://user-images.githubusercontent.com/21147581/148306722-be124433-eccc-4f66-b466-085c284fba2a.png" alt="drawing" width="700"/>

3. Enjoy the results!
